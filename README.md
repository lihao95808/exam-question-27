# 第27题考试网页

这是一个静态 HTML 展示页，用于展示初一年级数学期末考试第 27 题原创压轴题。

## 文件

- `index.html`：正式考试网页展示页
- `DOC/2026-07-05-第27题审题修订记录.md`：题目审查与修订记录

## 本地预览

直接用浏览器打开 `index.html` 即可，也可以运行：

```powershell
python -m http.server 8775 --bind 127.0.0.1
```

然后打开 `http://127.0.0.1:8775/`。
