# 七年级下册数学期末考试卷网页

本仓库公开展示武汉市硚口区七年级下学期数学期末模拟试卷。页面适合电脑、手机浏览和A4打印，公开文件仅包含学生作答所需的题面、图形与作答区域。

## 在线访问

- 最新试卷（二）：https://lihao95808.github.io/exam-question-27/
- 往期试卷（一）：https://lihao95808.github.io/exam-question-27/papers/set-a/

## 本地预览

直接用浏览器打开 `index.html`，或运行：

```powershell
python -m http.server 8775 --bind 127.0.0.1
```

然后访问 `http://127.0.0.1:8775/`。
