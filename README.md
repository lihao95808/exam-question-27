# 七年级下册数学期末考试卷网页

这是一个静态 HTML 试卷网页，用于对外展示一套七年级下册数学期末考试卷。公开页面只包含题面和作答区域。

## 文件

- `index.html`：正式试卷网页展示页

## 本地预览

直接用浏览器打开 `index.html` 即可，也可以运行：

```powershell
python -m http.server 8775 --bind 127.0.0.1
```

然后打开 `http://127.0.0.1:8775/`。
