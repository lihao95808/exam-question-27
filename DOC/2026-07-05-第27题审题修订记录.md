# 第27题审题修订记录

- 日期：2026-07-05
- 特性或问题：初一下期末压轴题坐标几何原创命题
- 命题结构：
  - 平移变换：由 `A(6,6)`、`B(2,2)` 向左平移 `m` 得到 `C(6-m,6)`、`D(2-m,2)`。
  - 坐标构图：第（1）问考查平移后坐标与水平底边三角形面积。
  - 交轴求参：第（2）问用直线 `AD` 与 `y` 轴交点 `P(0,3)` 建立关系，适合用相似三角形或斜率思想求参。
  - 面积参数：第（3）问通过动点 `M(a,a+2)` 构造与 `a` 无关的面积关系。
- 自审发现与修订：
  - 初稿若使用直线 `BD` 与 `y` 轴交点，由于 `B`、`D` 是水平平移对应点，`BD` 为水平线，交点不随 `m` 变化，条件退化。
  - 修订为“直线 `AD` 与 `y` 轴交于点 `P`”，此时条件 `OP=3` 可唯一确定 `m=4`。
  - 第（3）问使用“由 `A,C,D,B` 依次围成的平行四边形 `ACDB`”，避免按 `ABCD` 顺序造成顶点连接歧义。
- 关键验证：
  - 第（1）问：`m=2` 时，`C(4,6)`、`D(0,2)`；`AC=2`，点 `B` 到直线 `AC` 的距离为 `4`，所以 `S_ABC=4`。
  - 第（2）问：若 `P(0,3)` 在 `AD` 上，则用相似三角形有 `3:1=6:(m-2)`，解得 `m=4`；或由直线截距 `y=6-24/(m+4)=3` 得 `m=4`。
  - 第（3）问：`m=4` 时，`C(2,6)`、`D(-2,2)`，`0<a<4` 保证 `M(a,a+2)` 位于平行四边形内部；计算得 `S_ABM=4`、`S_CDM=4`，故存在常数 `k=1`。
